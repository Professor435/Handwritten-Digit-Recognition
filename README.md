# Handwritten-Digit-Recognition
I've created a **fully functional, visually stunning Handwritten Digit Recognition** system. Here's everything included:

---

## 🎯 Features Included

| Feature | Description |
|---------|-------------|
| **🎨 Drawing Canvas** | 280×280px responsive canvas with smooth drawing |
| **🔮 AI Prediction** | TensorFlow.js CNN model for real-time digit recognition |
| **📊 Probability Bars** | Visual breakdown of confidence for all digits 0-9 |
| **📁 Image Upload** | Drag & drop or click to upload images |
| **🖌️ Brush Size Control** | Adjustable brush thickness (5-30px) |
| **↩️ Undo/Redo** | Multiple undo levels with state saving |
| **📜 Prediction History** | Stores last 12 predictions with thumbnails & timestamps |
| **⌨️ Keyboard Shortcuts** | Space (Predict), C (Clear), Ctrl+Z (Undo) |
| **📱 Touch Support** | Full mobile/touchscreen compatibility |
| **⚡ Real-time Stats** | Inference time, confidence %, prediction count |
| **🎉 Particle Effects** | Celebration animation on prediction |
| **🍞 Toast Notifications** | Interactive feedback messages |
| **🌙 Dark Theme** | Modern glassmorphism UI with animated background |

## 🧠 Model Architecture

```
Input (28×28×1) → Conv2D(32) → MaxPool → Conv2D(64) → MaxPool → 
Flatten → Dense(128) → Dropout(0.2) → Dense(10, softmax)
```
